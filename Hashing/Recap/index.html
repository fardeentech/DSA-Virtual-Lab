Recap of Hashing

Overview

Hashing is a technique used for fast data retrieval by using a hash function to map data items to unique positions in a hash table. It is widely used in various applications where constant time complexity is desired for search, insertion, and deletion operations.

Key Characteristics

  • Type: Data retrieval technique using a hash table
  • Time Complexity:
    • Best Case: O(1) (for search, insert, delete)
    • Average Case: O(1) (with good hash function and minimal collisions)
    • Worst Case: O(n) (if all elements collide and are stored in a single bucket)
  • Space Complexity: O(n) (space required to store the hash table)
  • Efficient: Ideal for large datasets and fast lookup operations.
  • Handling Collisions: Various techniques such as chaining or open addressing can be used to resolve hash collisions.

How It Works

A hash function takes the input data (key) and converts it into a hash code (an integer). This hash code is then mapped to a location (index) in the hash table. If multiple keys map to the same index (collision), a collision resolution method is used to handle it.

Steps of Hashing

  1. Choose a hash function that distributes the keys uniformly across the hash table.
  2. Use the hash function to compute the hash code of the key.
  3. Store the data at the index corresponding to the hash code in the hash table.
  4. In case of a collision, apply a collision resolution technique such as chaining or open addressing.
  5. To search for an element, compute the hash code and check the corresponding index.

Example

Given keys: [23, 44, 12, 57, 89]

Assume a simple hash function is key % 10. After hashing, the keys will be stored at the following indices in the hash table: [12, 23, 44, 57, 89] (assuming no collisions).

Use Cases

Hashing is commonly used in:

  • Implementing hash-based data structures like hash maps and hash sets
  • Databases for fast key-based data retrieval
  • Caching systems to store frequently accessed data
  • Password storage with cryptographic hash functions
