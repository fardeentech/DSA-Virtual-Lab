Insertion Sort

Estimated Time

1 hour

Learning Objectives of the Experiment

  • Generate a sorted array from an unsorted array using Insertion Sort.
  • Optimize the Insertion Sort algorithm for better performance.
  • Demonstrate knowledge of time complexity by counting operations in each iteration.
  • Compare Insertion Sort with other sorting algorithms and understand its stability as a comparison sorting algorithm.
