Linked List

Estimated Time

1 hour

Learning Objectives of the Experiment

  • Implement a singly linked list and demonstrate basic operations such as insertion, deletion, and traversal.
  • Implement a doubly linked list and explain the differences between singly and doubly linked lists.
  • Demonstrate knowledge of memory management by properly handling node allocation and deallocation.
  • Compare linked lists with arrays in terms of memory usage, access time, and performance in different scenarios.
