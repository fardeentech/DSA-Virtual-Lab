Recap of Insertion Sort

Overview

Insertion Sort is a simple sorting algorithm that builds a sorted array (or list) one element at a time. It is much less efficient on large lists than more advanced algorithms such as quicksort, heapsort, or merge sort.

Key Characteristics

  • Type: Comparison-based sorting algorithm
  • Time Complexity:
    • Best Case: O(n) (when the array is already sorted)
    • Average Case: O(n²)
    • Worst Case: O(n²)
  • Space Complexity: O(1) (in-place sorting)
  • Stable: Yes, maintains the relative order of equal elements.
  • Adaptable: Efficient for small data sets and partially sorted arrays.

How It Works

The algorithm divides the input list into two parts: a sorted part and an unsorted part. Initially, the sorted part contains only the first element. The algorithm iterates through the unsorted part, picking each element and placing it in the correct position within the sorted part.

Steps of Insertion Sort

  1. Start with the second element (index 1) as the current element.
  2. Compare the current element with the elements in the sorted part (to its left).
  3. Shift all elements that are greater than the current element one position to the right.
  4. Insert the current element at its correct position.
  5. Repeat until the entire list is sorted.

Example

Given the array: [5, 2, 9, 1, 5, 6]

After applying Insertion Sort, the sorted array will be: [1, 2, 5, 5, 6, 9]

Use Cases

Insertion Sort is particularly useful for:

  • Small datasets
  • Partially sorted arrays
  • As a part of more complex algorithms (like Timsort)
